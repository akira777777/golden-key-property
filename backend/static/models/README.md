# Локальные 3D-модели недвижимости

## Как добавить свою модель

1. **Подготовьте файл модели** в формате `.glb` (рекомендуется) или `.gltf`.
2. **Поместите файл** в папку `backend/static/models/`.
3. **Укажите путь** в поле `tour_url` в базе данных, например `/static/models/villa.glb`.
4. **Установите тип тура** `MODEL_3D`.

## Пример обновления через SQLite

```sql
UPDATE listings
SET tour_type = 'MODEL_3D',
    tour_url = '/static/models/villa.glb'
WHERE id = 1;
```

Или добавьте при инициализации в `main.py` в список `listings`.

## Где взять .glb модели

- **Sketchfab** — экспорт моделей в `.glb`.
- **Blender** — File → Export → glTF 2.0 (.glb).
- **Spline** — экспорт в `.glb` для веба.
- **Matterport** — конвертация сканов в .glb через сторонние сервисы.

## Оптимизация

- Размер файла: желательно до 20 МБ для быстрой загрузки.
- Полигоны: до 500K полигонов для плавного рендера.
- Текстуры: используйте сжатие WEBP/KTX2, встроенные в glTF.
- Draco compression уменьшает размер в 5-10 раз.

## Использование в UI

Фронтенд автоматически определяет локальную модель по расширению `.glb` и рендерит её через `<model-viewer>`:

- Перетаскивание мышью — вращение камеры.
- Колёсико мыши — приближение/отдаление.
- Двойной клик — автоматическое вращение вкл/выкл.
- Кнопка AR — просмотр в дополненной реальности (на мобильных).
